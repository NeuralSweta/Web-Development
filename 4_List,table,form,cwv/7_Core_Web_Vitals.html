SEO: Search Engine Optimization(SEO) refers to the practice of optimizing a website's HTML code to improve its visibility in search engine results.

HTML developers can implement SEO using the following techniques: 1. Title Tag: 2. Meta Description: summary of webpage. 3. URL Slug:A URL slug is the last part of a URL address that serves as a unique identifier of the page. 4. Meta Author Tag: 5. Favicon: a small icon that represents your website 6. Image Optimization: Compress images to improve page load times. 7. Optimize Resources: Remove unused HTML, CSS, and JavaScript files. 8. Semantic HTML: header,nav,article,footer

CLS: Cumulative Layout Shift (CLS) is a metric that measures the visual stability of a webpage as it loads. A good CLS score is 0.1sec or lower.

INP:Interaction to Next Paint (INP) a web performance metric that measures the time between a user's first interaction with a webpage—such as clicking a link or tapping a button—and the moment the browser responds. FID(First input delay) replaced by INP.
