CSS Varibles...

CSS variables (also called custom properties) are reusable values defined with a name, starting with --, that you can use throughout your CSS. They make it easy to update and manage colors, sizes, and other values in one place.
Here are some common terms for which you can create CSS variables:
Colors (e.g., --primary-color, --secondary-color, --text-color)
Font sizes (e.g., --heading-size, --body-size)
Spacing (e.g., --padding, --margin, --gap)
Border radius (e.g., --button-radius)
Z-index values (e.g., --modal-z)
Box shadows (e.g., --card-shadow)
