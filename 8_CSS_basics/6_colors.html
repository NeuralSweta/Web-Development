Colors Link

Colors can be represented in:

  1. Color Keyword: Uses predefined color names like red,blue, or green.
  2. Hex Color Code: Represents colors with a hexadecimal value, e.g., #ff0000 for red.
  3. RGB: Specifies color using Red, Green, and Blue values, e.g.,rgb(255,0,0) for red.
  4. RGBA: Like RGB, but adds an Alpha (opacity) value, e.g.,rgba(255,0,0,0.5).
  5. HSL: Uses Hue, Saturation, and Lightness to define color, e.g., hsl(0, 100%, 50%)for red.
